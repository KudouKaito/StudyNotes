session

session

tmux的命令可以不打全，只要能匹配到唯一命令即可，比如tmux a即可匹配到tmux attach。类似ip a匹配ip address

新建session

tmux new -s <session-name>

离开session

tmux detach

查看session列表

tmux ls
tmux list-session

Ctrl+b s

恢复session

tmux attach
# 通常
tmux a

tmux attach -t <session-name>
